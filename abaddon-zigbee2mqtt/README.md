# Abaddon Zigbee2MQTT Home Assistant Add-on

Custom build of Zigbee2MQTT with personalized features and enhancements.

## Installation

1. Add this repository to Home Assistant:
   - Go to **Settings** → **Add-ons** → **Add-on Store**
   - Click the **⋮** menu (top right) → **Repositories**
   - Add: `https://github.com/abaddon/zigbee2mqtt`

2. Install the add-on:
   - Refresh the Add-on Store page
   - Find "Abaddon Zigbee2MQTT" in the list
   - Click on it and click **Install**

3. Configure the add-on:
   - Go to the **Configuration** tab
   - Set your MQTT server details
   - Set your serial port device
   - Save and start the add-on

## Configuration

See the **Documentation** tab in the add-on for full configuration options.

## Features

- Custom Zigbee2MQTT build
- Automatic updates from custom repository
- Integration with Home Assistant
