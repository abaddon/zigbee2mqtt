# Add-on Icons

To add icons for your add-on, place the following files in this directory:

## Required Files

### icon.png
- **Size**: 256x256 pixels
- **Format**: PNG with transparency
- **Purpose**: Shown in the add-on list

### logo.png
- **Size**: 256x256 pixels (minimum)
- **Format**: PNG with transparency
- **Purpose**: Shown on the add-on details page

## Optional

You can use the official Zigbee2MQTT logo or create your own custom logo.

To get started, you can:
1. Download the Zigbee2MQTT logo from: https://www.zigbee2mqtt.io/images/logo.png
2. Create a custom variant with your branding
3. Place both `icon.png` and `logo.png` in the `abaddon-zigbee2mqtt/` directory

## Without Icons

The add-on will work perfectly fine without icons - they're purely cosmetic.
